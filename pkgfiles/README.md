This directory contains files or file templates for use with package managers:

- `gentoo/tek-steamclient.ebuild` - Gentoo ebuild file template, used by [tek-overlay](https://github.com/teknology-hub/tek-overlay)
- `msys2/PKGBUILD` - Package build script template (with unset pkgver) for [MSYS2](https://msys2.org/), intended to be used in [TEK](https://github.com/teknology-hub/MinGW-env) environment. Used by `mingw-w64-tek-steamclient` package in [TEK MinGW package repository](https://github.com/teknology-hub/MINGW-packages)
